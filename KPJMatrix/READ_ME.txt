To run this LU Decomposition program:
(1) open your terminal and change directory to this directory.
(2) run "javac KPJMatrix.java"
(3) open input.txt file and put your desire matrix. If the matrix is singular, an error might occur.
(4) now run "java KPJMatrix < input.txt".
(5) enjoy Decomposing.

NOTE:
KPJMatrix is a matrix abstraction, supports multiply and plus, decompose upper/lower triangular matrices.
